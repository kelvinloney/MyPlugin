== Description ==

This plugin allows the user to customize their login page and admin area.

The customization options are as follows:

Customize Login Page
*   Custom URL
*   Custom Title
*   Custom Style
*   Custom Message

Customize Admin Area
*   Custom Footer
*   Custom Toolbar
*   Custom Scheme



== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/myplugin` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->MyPlugin screen to configure the plugin
